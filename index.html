Ever have trouble recalling the exact syntax for your favorite CSS code? Give it a permanent home and add it to this page! Select any snippet below and it'll automatically select all of the code for you to copy.

FlexBox Row

Use these preoperties to create a flexbox row layout

        .row { display:flex;

        flex-direction : row;

        flex-wrap: wrap; }

FlexBox Column

Use these properties to create a flexbox Column layout

        .column { display:flex;
        flex-direction : column;
        }

CSS Grid layout

Build a !2- Column using CSS grid

        .grid { display:grid;

        width:100% ; 

        grid-templates-column: repeat(12,1fr); }
